## Что было сделано

### 1. Авто-экипировка (НОВОЕ)

Бот теперь автоматически надевает всё лучшее снаряжение при входе в PvP:

```
✅ Нетеритовая броня (шлем, нагрудник, штаны, ботинки)
✅ Щит в левую руку (off-hand)
✅ Лучшее оружие в правую руку (меч/топор)
```

**Проблема:** Бот бил щитом вместо меча, не надевал полный сет брони.

**Решение:**
- Асинхронная экипировка с правильным порядком (броня → щит → оружие)
- Принудительная проверка: если в руке не меч — меняем на меч
- Все операции через `await` для корректной последовательности

---

### 2. Система хила (УЛУЧШЕНО)

#### Приоритеты хила на близкой дистанции (< 6 блоков):
1. 🧪 **Splash Potion** (мгновенный хил)
2. 🍎 **Enchanted Golden Apple** 
3. 🍏 **Golden Apple**

#### Критический режим (HP ≤ 6 сердец):
- Кулдаун хила: **500мс** (вместо 1500мс)
- Бот **не атакует**, только хилится
- **Не прерывает еду** даже если бьют
- Автоматически переходит в режим отступления (KITE)

**Проблема:** Бот не хилился splash зельями, прерывал еду чтобы бить, умирал на критическом HP.

**Решение:**
- Исправлена проверка NBT у splash зелий
- Флаг `_isEating` — пока ест, не атакуем
- Форсированный heal при HP ≤ 6

---

### 3. Защита щитом (УЛУЧШЕНО)

- Щит всегда в левой руке во время боя
- Щит не трогается пока бот ест (не прерывает хил)
- После еды автоматически возвращает щит

**Проблема:** Щит был в правой руке, бот бился щитом, щит прерывал процесс еды.

**Решение:**
- Метод `_ensureShieldEquipped()` — проверяет щит каждые 500мс
- Проверка `_isEating` во всех shield-операциях

### 5. После боя (УЛУЧШЕНО)

При убийстве цели:
1. Если HP < 20 → хилится (использует доступные предметы)
2. Через 2 секунды → останавливается
3. Голосовая фраза (если настроена)

**Проблема:** Бот не останавливался после убийства, бежал искать цель дальше.

**Решение:** Логика в `_tick()` — проверка HP цели и автоматический stop.

---

### 6. Команда "стоп" (ИСПРАВЛЕНО)

**Проблема:** Бот не реагировал на команду "стоп" во время PvP.

**Решение:** Подписка на `MovementEvents.SET_IDLE` в `CombatSystem.js` — при любой команде остановки выходим из PvP.

## Как использовать

### Вход в PvP:
```
атаковать меня
атаковать [ник игрока]
```

### Выход из PvP:
```
стоп
```

### Требования к инвентарю:
- Броня (приоритет: нетерит → алмаз → железо)
- Оружие (меч/топор)
- Щит (в off-hand)
- Хил: splash potions + golden apples

## Конфигурация

В `config.js`:

```js
pvpEnabled: true              // Включить PvP
pvpIdealDistance: 2.9         // Идеальная дистанция
pvpAttackCooldown: 600          // Кулдаун атаки (мс)
pvpKiteHpThreshold: 8         // Порог для отступления
pvpTradeHpThreshold: 4        // Порог для режима обмена ударами
```
